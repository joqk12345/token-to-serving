# Chapter Brief

For the named chapter slug:

1. Read `book/spine.yml`.
2. Read available source notes under `book/evidence/source-notes/`.
3. Use `.claude/docs/chapter-template.md`.
4. Create or update `book/chapters/<slug>-brief.md`.
5. Update `book/STATUS.md` if the brief clears.

Do not draft the chapter body in this command.
