  o E.g. TensorFlow and PyTorch.

• It optimizes inference for multiple query types.
  o E.g. real-time, batch, and streaming.

• It can be deployed on different environments
  o E.g. public cloud and embedded devices.

                       LightLLM
• It performs tokenization, model inference, and
  detokenization asynchronously, to improve GPU utilization.

• It implements Token Attention, a token-wise's KV cache
  memory management algorithm.

• It adopts Efficient Router scheduling implementation, which
  collaborates with Token Attention to manage the GPU
  memory of each token.
