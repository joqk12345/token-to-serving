LLM 量化基础

这是一篇面向大学新生的导学文档，根据课程课件重新组织。原始 PDF 仍保留在文末，供深入阅读。

学习目标

  • 理解量化解决的资源问题
  • 认识 bit 宽度和误差
  • 知道量化不等于免费加速

核心解释

量化用更少 bit 表示权重或激活，常见目标是减少显存占用和内存带宽压力。

低 bit 会引入近似误差，需要检查模型质量是否还能接受。

实际速度提升取决于硬件和 kernel 是否支持低 bit 高效计算。

课堂例子

把高清照片压缩成较小文件可以省空间，但压缩太狠会失真。量化也有类似的精度和资源权衡。

检查问题

  1. 量化主要节省什么资源？
  2. 为什么 4-bit 模型不一定总比 16-bit 快？
